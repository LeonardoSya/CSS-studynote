flex布局又称弹性布局 通过给父盒子添加flex来控制子盒子排列方式和位置 任何容器都能指定为flex布局 父盒子flex后 子元素的float、clear、vertical-align失效 采用flex布局的元素 称为flex容器 它的所有子元素成为 flex item (项目) 主轴(默认是x轴)侧轴(默认是y轴) 但不是绝对 父项属性: 主轴方向 : row(默认 从左到右) / row-reverse(右到左) / column(上到下) / column-reverse(下到上) 主轴上 子元素排列方式 (使用这个属性前 要先确定好主轴是哪个) : flex-start(默认 从头部开始) / flex-end(从尾部开始) / center(在主轴居中对齐) / space-around(平分剩余空间 不合并外边距) / space-evenly(每个元素之间间隔相等 合并外边距) / space-between(先两边贴边 再平分剩余空间) 默认情况下项目都排在一条轴线上 不换行 如果装不开会缩小子元素宽度 ; : nowrap(默认 不换行) / wrap(换行) 侧轴上 子元素排列方式 (单行) : center(挤在一起居中) / flex-end(下到上) / flex-start(从上到下) / stretch(拉伸) 侧轴上 子元素排列方式 (多行) 当需要换行的时候会激活"多行" 否则单行下没有效果 : flex-start(默认 在侧轴头部开始排列) / flex-end(在侧轴尾部开始排列) / center(侧轴中间显示) / space-around/space-evenly/space-between / stretch(子元素高度平分父元素高度) 简写形式 复合属性 相当于同时设置flex-direction和flex-wrap eg. flex-flow: column wrap; 子项属性： 从剩余空间分配 用flex表示占多少份数 .item { flex: ; } 允许单个项目与其他项目不一样的对齐方式 覆盖align-items : auto(默认 表示继承父元素的align-items 若没有父元素则=stretch) 其他值等同于align-items 项目排列顺序 数值越小 排列越靠前 默认0 可负数
